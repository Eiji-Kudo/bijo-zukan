## はじめに

### 前提条件

- Node.js (v15 以上推奨)
- npm

### インストール

1. リポジトリをクローンする

    ```
    git clone https://github.com/your-username/bijo-zukan.git
    cd bijo-zukan
    ```

2. 依存関係をインストールする

    ```
    npm install
    ```

### ローカルでの起動

    npm run dev

ブラウザで http://localhost:3000 を開くとアプリが表示されます。
